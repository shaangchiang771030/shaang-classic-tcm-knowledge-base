# 四象時間模型（陰陽互含規律） (Four Images Temporal Model)

純粹的二分法（陰與陽）雖然工整，但在連續的時間流體中，事物在陰陽轉換時必然存在過渡狀態。這便催生了「四象時間模型」。
四象的核心本質是**「陰陽互含規律」即少陽（陰中之陽）、太陽（陽中之陽）、少陰（陽中之陰）、太陰（陰中之陰）。它是中醫捕捉時間動態轉折點的高階利器。也可理解為, 完整周期被區分為四個相位區間.

在時間的演進中，陽氣的誕生並非一蹴而就，而是經歷了「陰中生陽（春/黎明） -> 陽氣極盛（夏/正午） -> 陽中生陰（秋/黃昏） -> 陰氣極盛（冬/半夜）」的循環。在閱讀經典時，只要看到「陰中有陽、陽中之陽」等描述，指的就是四象模型。
這裡把公轉與自轉的對應相位區間,一組地描述, 是為了強化同一組能量符號,在不同時間尺度的靈活運用. 

四象模型在《黃帝內經·四氣調神大論》中被用來指導日常作息。例如，冬天（太陰）的養生原則是「早臥晚起」，這本質上是利用年尺度的四象規律去對齊、指導單日之內日尺度的四象作息。網站主的個人壞習慣：午後喝咖啡。黃昏後天時已進入少陰（陽中之陰），人體能量本應放緩、冷卻以進入黃昏狀態；此時喝咖啡等於強行在體內「點起一個太陽」，這是一種嚴重違反天時四象時間模型的「逆天行為」。
