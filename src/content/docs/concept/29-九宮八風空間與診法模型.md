# 九宮八風空間與診法模型 (Nine Palaces and Eight Winds Model)

在天人之學的巨大族譜中，除了橫跨天時變化的運氣學說，還有一套專注於探討「地之空間規律」的特殊模型，這就是「九宮八風空間與診法模型」。這套模型利用工整的 3 × 3 矩陣，將大地的幾何幾何空間、氣流方向與人體的九州九野分布進行了高度抽象化的拓撲綁定。

它的源頭就是洛書. 九宮格是古人在二維及三維平面上進行位置標定與能量動態推算的最簡完美矩陣。在人體空間上，它將上、中、下與天、地、人縱橫相乘，形成九個拓撲區塊（九宮）；在外部環境上，它對應著從八個幾何幾何方向吹來的「八風」能量。這是一套專門用來捕捉地氣影響力的空間診法。

在經文中，本模型直接催生了《黃帝內經·素問》中極其古老且權威的「三部九候診法」，將人體頭、手、足各分天地人，構成九個精密診察候區。當代中醫師董乾陽先生在其著作《九宮圓運動》中的深入挖掘, 將人體各解剖部位以及中藥「入藥部位」完美安進了九宮幾何幾何模型中。例如「花類藥材」之所以屬火、屬南方、對應九宮格的上方，第一性原理就在於植物在夏天（火）最易觸發開花機制，因而花朵封裝了自然界最向外宣散的火熱能量。這證明九宮模型是指導臨床本草入藥與空間診法的完美工具。
