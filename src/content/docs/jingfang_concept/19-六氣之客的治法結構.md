《至真要大論》這一組「六氣之客」條文，討論的不是某一氣已經勝出，也不是勝後必復，而是六氣以「客氣」身分加臨主位時，如何在主客關係中安置補、瀉與後續功能操作。形式化驗證後，最重要的結論是：「某氣之客」第一列所列之味，恰好都是補客氣本身的味；第二列所列之味，則恰好都是瀉客氣本身的味。

也就是說，厥陰木客用辛補、酸瀉；少陰火客用鹹補、甘瀉；太陰土客用甘補、苦瀉；少陽火客同少陰，用鹹補、甘瀉；陽明金客用酸補、辛瀉；太陽水客用苦補、鹹瀉。這六組完全符合五行補瀉正味表，程式驗證為 6/6。

因此，「厥陰之客，以辛補之，以酸寫之」這類句子，不應被理解成辛一定永遠作君味，酸一定永遠作臣味。它更像是一張「客氣補瀉詞典」：當這一客氣不足、受主氣壓制時，可取其補味；當這一客氣太過、反壓主位時，則取其瀉味。真正臨床判斷仍要先看主客勝負，而不是看見「以辛補之」就無條件補客。

這也是「六氣之客」與一般「六氣之勝、六氣之復」最重要的差異。一般勝復有「有勝則復」的補償反彈邏輯；但客主之氣，原文另有「勝而無復」「主勝逆，客勝從」「有餘折之，不足補之，必安其主客」的框架，所以客主勝負不能直接套入一般勝復模型。

所以，這組「六氣之客」最精簡的治法結構可以寫成：

六氣之客：
客氣加臨主位
→ 第一列味 = 補客氣之味
→ 第二列味 = 瀉客氣之味
→ 實際用補或用瀉，取決於主勝或客勝

客勝：
瀉客 + 補主

主勝：
瀉主 + 補客

主客平：
不因主客關係額外干預

後面的「以甘緩之」「以鹹收之」「以甘緩之」「以鹹耎之」「以苦泄之」「以苦堅之、以辛潤之」則屬於尾巴功能算子，不應再被看成普通的第三味、第四味。它們是在補瀉客氣之後，進一步指定氣化操作方向：甘緩、鹹收、鹹耎、苦泄、苦堅、辛潤。尤其太陽之客最後說「開發腠理，致津液通氣也」，正好說明苦堅與辛潤不是抽象配味，而是為了打開腠理、使津液與氣機重新流通。

因此，這組條文的真正價值，不在於背誦「厥陰辛酸、少陰鹹甘、太陰甘苦、少陽鹹甘、陽明酸辛、太陽苦鹹」，而在於看出《內經》對「客氣」的處理方式：先建立客氣自身的補瀉詞典，再根據主客勝負決定用補或用瀉，最後以尾巴算子完成緩、收、耎、泄、堅、潤等具體氣化操作。 這使「六氣之客」成為一套清楚可計算的主客調平語法，而不是零散的用味經驗表。
