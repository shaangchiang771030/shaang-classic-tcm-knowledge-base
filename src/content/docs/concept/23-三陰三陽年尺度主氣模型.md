# 三陰三陽年尺度主氣模型 (Three Yin Three Yang Host Qi Model)

與代表隨機擾動的「客氣」模型相對立且完美正交的，便是「三陰三陽年尺度主氣模型」。
主氣模型在天人之學的框架中，牢牢守護著常變原則中的那個恆常基線——「常」。
它是大地氣候每年周而復始、永恆不變的基本自然秩序。
主氣，且在一年之內切分成六個時段（初之氣到終之氣，各管60天）。其運轉的嚴格符號順序為：厥陰, 少陰,少陽, 太陰, 陽明, 太陽.
其起始點有著極其嚴格的科學規定：必須精準起始於「大寒」節氣。
本模型的經典依據遍布經典，如《黃帝內經·素問·六節藏象論》中明言：「帝曰：願聞六六之節……岐伯曰：周天氣者，六期為一備；折指南針，六復為一歲。」這裡的六六之節（10×6=60周甲，60×6=360日）精準描述的便是主氣模型的年尺度輪轉。
