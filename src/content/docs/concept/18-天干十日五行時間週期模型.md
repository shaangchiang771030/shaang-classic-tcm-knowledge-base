# 天干十日五行時間週期模型 (Heavenly Stems 10-Day Five Elements Model)

五行模型不僅能放大到一年360天的宏觀尺度（各管72日），更能縮小到以天為單位的微觀時間週期，這就是「天干十日五行時間週期模型」。它是古人在時間維度上，利用十個天干符號來精細捕捉五行能量微觀波動的架構。

本概念的核心第一性原理在於「十進制天體能量共振週期的拆分封裝」。古人將十天干（甲、乙、丙、丁、戊、己、庚、辛、壬、癸）兩兩一組，拆分成五組相鄰的陰陽對子（甲乙屬木、丙丁屬火、戊己屬土、庚辛屬金、壬癸屬水），每組各管兩天，在十天之內完成一次完美的五行小循環。這套十日週期是古代時間運算中不可或缺的函數參數。但要特別小心的是, 甲與乙雖同属木, 但陰陽属性不同. 在古中醫
的音樂學裡, 甲與乙代表了同為角音的高低不同頻率狀態. 因此, 流傳千年的「子午流注」針法，其十干支所對應的開穴順序，亦是深植於這個十日循環之中。
