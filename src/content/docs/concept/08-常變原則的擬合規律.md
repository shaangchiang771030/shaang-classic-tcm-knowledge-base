# 常變原則的擬合規律 (Constancy and Change Principle)

在群星運轉與萬物氣化中，自然規律呈現出雙重特性：一方面有年年如常的恆常週期，另一方面又有氣候異常、每年不同的變動因素。中醫時空模型的核心智慧之一，便是「常變原則」。常變原則要求在建構模型時，必須設定一個代表基本秩序的「恆常主軸（常）」,英文可稱為Baseline，再加上一個捕捉動態擾動的「偏差量」，,英文可稱為Deviation，兩者結合才能對複雜現象進行完美的擬合。
這與現代資料科學中處理非線性信號的方法如出一轍。若只看「常」，模型會流於僵化，無法解釋暖冬或春寒等異常；
若只看「變」，則會陷入無序的混亂。透過「主氣（常）」與「客氣（變）」的疊加，古人成功用一個工整的方程式，運算出動態氣象對生理產生的複雜干預。
