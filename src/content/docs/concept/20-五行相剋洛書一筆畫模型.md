# 五行相剋洛書一筆畫模型 (Five Elements Restriction Model)

在五行的動態交互中，「相生」與「相剋」常被通俗地畫在一起。相剋模型因其對疾病傳變與制約秩序的決定性影響，必須單獨提取出來深入解構，這就是「五行相剋洛書一筆畫模型」。這是一個將神祕的洛書數理與空間相剋秩序進行完美對接的硬核數學模型。
五行相剋（水剋火、火剋金、金剋木、木剋土、土剋水）的本質，是維持系統生態平衡的負回饋機制。令人震撼的是，這條相剋鏈條的幾何軌跡，完美地封裝在中國上古聖物——「洛書」的數理九宮格之中。
將五行的「生數」與「成數」標上它們該有的五行顏色，然後填入洛書的九宮格中。此時若使用「一筆畫法（Single-Stroke Drawing）」，假設我們從北方（1, 6）的水出發，下一站便是水所剋的火——也就是右上角南方（2, 7）的火；再下一站便是火所剋的金——西方（4, 9）的金；接著是金所剋的木——東方（3, 8）的木；隨後是木所剋的土——中央（5, 10）的土；最後再回到北方水。
